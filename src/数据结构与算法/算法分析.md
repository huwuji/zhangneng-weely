1. 运行事件分析：
类型：O(N),O(N^m),O(logN)

2. 运行时间中的对数------O(NlogN)
具有对数特点算法的栗子：
1）对分查找：二分查找，（ps：归并排序算法也是如此）

2）欧几里得算法：即计算最大公因数的一种算法。（这个比较重要，需要记住。）
直接上代码：
```
function getHCF(n,m){
    let rem=null;
    while(rem!==0){
        if(n>=m){
            rem=n%m;
            n=m;
            m=rem;
        }else{
            rem=m%n;
            m=n;
            n=rem;
        }
    }
    return Math.max(n,m);
}
```
